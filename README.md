# ARB LegalBot Web App

A hybrid legal research assistant for California and Federal environmental law.

## Features

- Chat + search interface
- Topic filters (CEQA, Clean Air Act, etc.)
- Dark theme with ARB LegalBot branding
- SharePoint-compatible static web app

## How to Deploy on SharePoint

1. Upload all files to a SharePoint document library or static site.
2. Use a SharePoint web part or embed the `index.html` in a page.
3. Optionally connect to a backend API for real-time legal answers.
